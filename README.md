This project is a static frontend clone of the Amazon Prime Video login page built using HTML and CSS.

The main objective of this project is to practice frontend development by recreating a real-world website interface. It focuses on replicating the original UI design, improving HTML structure, applying CSS styling, maintaining proper layout alignment and spacing, and understanding the basics of responsive web design.
